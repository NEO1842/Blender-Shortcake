# Contributing

Thanks for your interest in this project!

- Feel free to open an Issue for bugs or suggestions.
- Pull requests (model fixes, material tweaks, documentation updates) are welcome.
- Please test your changes in Blender 3.4 before submitting.

--日本語--

# コントリビューションガイド

このプロジェクトに興味を持っていただきありがとうございます！

- 不具合や提案がある場合は、お気軽にIssueを立ててください。
- プルリクエスト（モデルの修正、マテリアルの調整、ドキュメント更新など）も歓迎します。
- 提出前にBlender 3.4で動作確認をお願いします。

--English--

# Security Policy

This project contains Blender scene/asset files (models, materials, textures) and no executable code.

If you discover a security concern (e.g. a suspicious script embedded in a file), please report it privately using GitHub's [private vulnerability reporting](https://docs.github.com/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability) instead of opening a public issue.

--日本語--

# セキュリティポリシー

このプロジェクトにはBlenderのシーン・アセットファイル（モデル、マテリアル、テクスチャ）のみが含まれており、実行可能なコードは含まれていません。

セキュリティ上の懸念（ファイルに埋め込まれた不審なスクリプトなど）を発見した場合は、公開のIssueではなく、GitHubの[プライベートな脆弱性報告機能](https://docs.github.com/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability)を使って非公開で報告してください。
